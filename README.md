## Overview

This is the official website for **Medeena English Center**, built with **Next.js** and **TypeScript**. The platform serves as a hub to showcase English programs, registration details, educational resources, participant testimonials, and general information about the institution.

The main goal of the website is to provide prospective students with an **intuitive, responsive, and informative experience**, making it easier to explore courses, access promotions, and understand the offerings of Medeena English Center.

---

## Features

- Browse and explore all English courses available.
- Detailed course information including duration, level, and pricing.
- Online registration integrated via WhatsApp.
- Teacher profiles and educational background.
- Responsive design for both desktop and mobile.
- Smooth horizontal scrolling for featured classes.

---

## Tech Stack

- **Frontend:** Next.js, TypeScript, Tailwind CSS  
- **Deployment:** Vercel  
- **Styling & Animations:** Tailwind utilities & custom components
